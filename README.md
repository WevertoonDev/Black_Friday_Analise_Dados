# 🛍️ Análise de Dados - Black Friday

## 📌 Sobre o projeto

Este projeto tem como objetivo analisar dados de vendas da Black Friday utilizando SQL e SQLite.

Durante a análise, são realizadas consultas para extrair informações relevantes, responder perguntas de negócio e praticar comandos SQL aplicados à análise de dados.

Este projeto foi desenvolvido como parte dos meus estudos em SQL e Análise de Dados.

---

## 🎯 Objetivos

- Praticar consultas SQL.
- Analisar dados de vendas da Black Friday.
- Desenvolver habilidades em análise de dados.
- Construir um projeto para portfólio.

## 📈 Possíveis análises

- Total de vendas.
- Média de compras por cliente.
- Produtos mais vendidos.
- Categorias com maior faturamento.
- Perfil dos consumidores.
- Distribuição por idade e gênero.

---

## 🚀 Como executar

1. Clone este repositório.
2. Abra o banco de dados no SQLite Online ou DB Browser for SQLite.
3. Execute as consultas presentes no arquivo `consultas.sql`.
